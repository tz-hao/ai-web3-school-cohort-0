## 二、逐项数据流表

### 1. 用户输入

| 属性 | 值 |
|------|-----|
| **内容示例** | 「帮我在 Uniswap 用 100 USDC 换 ETH」 |
| **存储位置** | 本地日志文件（`~/.agent/logs/`） |
| **发给第三方** | ✅ LLM Provider（作为 prompt 的一部分） |
| **是否脱敏** | 不脱敏——这是核心任务描述，脱敏后 LLM 无法理解 |
| **用户可删除** | ✅ 删除本地日志文件 |

### 2. 钱包地址

| 属性 | 值 |
|------|-----|
| **内容示例** | `0xUserEOA...` |
| **存储位置** | 本地 Agent 配置（`~/.agent/config.json`）、链上（公开） |
| **发给第三方** | ✅ RPC Node（查询余额、交易历史） |
| **是否脱敏** | ⚠️ 可部分脱敏：发给 LLM 时可截断为 `0xUser...a1b3`。但发给 RPC 必须是完整地址 |
| **链上可见性** | 地址是公开的——任何人可以通过区块浏览器查到该地址的全部交易 |
| **用户可删除** | ❌ 链上记录不可删除。✅ 可删除本地配置 |

### 3. 交易历史

| 属性 | 值 |
|------|-----|
| **内容示例** | `[{"hash": "0xabc...", "from": "0xUser...", "to": "0xContract...", "value": "100 USDC", ...}]` |
| **存储位置** | 链上（公开）、本地缓存（`~/.agent/cache/tx_history/`）、索引层（The Graph / Dune） |
| **发给第三方** | ✅ RPC Node（查询）→ 发给 LLM（摘要后：最近 5 笔交易的关键信息） |
| **是否脱敏** | ✅ 发 LLM 时只发摘要：「最近 5 笔：2 笔 swap、1 笔 approve、2 笔转账」。不送原始 tx 列表 |
| **链上可见性** | 交易历史是公开的——每笔交易都在链上 |
| **用户可删除** | ❌ 链上记录不可删除。✅ 可删除本地缓存 ✅ 可取消索引层订阅 |

### 4. API Key

| 属性 | 值 |
|------|-----|
| **内容示例** | `sk-ant-api03-xxxxxxxx...` |
| **存储位置** | 系统环境变量（`AIW3_ANTHROPIC_KEY`）、本地加密 keystore |
| **发给第三方** | ✅ LLM Provider（作为 HTTP Authorization header）。❌ 不发给 RPC、IPFS、其他任何方 |
| **是否脱敏** | 不可能脱敏——必须原样发送 |
| **链上可见性** | ❌ 绝对不在链上 |
| **用户可删除** | ✅ 立即——删除环境变量 + keystore entry + 在 Provider 端 revoke key |

### 5. 模型上下文

| 属性 | 值 |
|------|-----|
| **内容示例** | 系统 prompt + 用户指令 + 链上状态摘要 + 最近对话 + 文档内容 |
| **存储位置** | 本地 Memory（`~/.agent/memory/`）、Provider 端（仅当次推理，不持久化） |
| **发给第三方** | ✅ LLM Provider（每次推理的完整上下文） |
| **是否脱敏** | ✅ 地址截断、金额脱敏（`100 USDC` 而非精确到 wei 的数值）、文档标记为 `<UNTRUSTED_CONTEXT>` |
| **敏感字段处理** | 私钥 ❌ 不进入上下文。API key ❌ 不进入上下文。用户真实姓名/邮箱 ❌ 不进入上下文（如果不需要） |
| **用户可删除** | ✅ 删除本地 memory 文件。⚠️ Provider 端——取决于 Provider 的隐私策略（选 zero-retention 的 API） |

### 6. 日志

| 属性 | 值 |
|------|-----|
| **内容示例** | `[2026-05-24 10:00] 用户请求: 查询余额 → RPC 调用: eth_getBalance → 返回: 1.5 ETH → 下一步: 显示给用户` |
| **存储位置** | 本地日志文件（`~/.agent/logs/`）、链上 Event（`AgentAction` event） |
| **发给第三方** | ❌ 本地日志不发给任何方。✅ 链上 Event 是公开的 |
| **是否脱敏** | ✅ 本地日志：地址截断 `0xUser...a1b3`。链上 Event：只存操作类型 hash `keccak256("BALANCE_QUERY")`，不存参数 |
| **链上可见性** | 链上 Event 公开可见，但只存操作类型 hash，不泄漏具体地址和金额 |
| **用户可删除** | ✅ 删除本地日志。❌ 链上 Event 不可删除 |

### 7. 记忆

| 属性 | 值 |
|------|-----|
| **内容示例** | `用户偏好: 网络 Base、滑点 0.5%、常用 Uniswap。上次学习: Agent Wallet 章节` |
| **存储位置** | 本地向量数据库（`~/.agent/memory/vectordb/`）、本地 SQLite（`~/.agent/memory/prefs.db`） |
| **发给第三方** | ❌ 完整记忆不发。✅ 相关记忆片段送入 LLM 上下文（RAG 检索：取出最相关的 3 条） |
| **是否脱敏** | ✅ 存入记忆前：地址脱敏、金额脱敏。只存偏好和模式 |
| **用户可删除** | ✅ 删除本地向量 DB + SQLite 文件，或单条删除 `agent memory forget "上次在 Uniswap 的滑点设置"` |

## 三、数据脱敏策略汇总

| 原始数据 | 脱敏方式 | 发给谁 | 示例 |
|----------|----------|--------|------|
| 完整地址 | 截断为前后各 6 位 | LLM Provider | `0xUser...a1b3` |
| 交易金额 | 四舍五入到 2 位小数 | LLM Provider | `100.00 USDC` 而非 `100000000` |
| tx hash | 保持完整 | RPC、区块浏览器 | 需要精确查询 |
| 用户姓名/邮箱 | 不采集、不存储 | 任何方 | — |
| API key | 加密存储，仅 HTTP header | LLM Provider | 绝不进入上下文或日志 |
| 私钥 | 永不进入 Agent 运行时 | 任何方 | Smart Account / MPC 方案管理 |
| 合约地址 | 保持完整 + 附 explorer 链接 | LLM Provider | 需要精确查询 |
| IP 地址 | 不记录（或 hash 后存） | 不发给任何方 | `sha256(ip)` 用于限流 |

### 撤销

```
用户命令: agent permission revoke

可撤销:
  ✅ Session Key → 调用智能账户撤销
  ✅ API Key → 在 Provider 控制台 revoke
  ✅ Token Allowance → 调合约 revoke
  ✅ Indexer 订阅 → 取消 The Graph / Dune 查询
  ⚠️  链上数据 → 不可撤销（已写入即永久）
```

### 删除

```
用户命令: agent data delete

可删除:
  ✅ 本地日志: agent data delete logs --before 2026-05-01
  ✅ 本地记忆: agent data delete memory --topic "uniswap"
  ✅ 本地缓存: agent data delete cache --all
  ✅ IPFS 文件: 如果 pin 在自己节点 → 可 unpin（但已传播的副本可能仍存在）
  ❌ 链上数据: 永久不可删除
  ❌ RPC 日志: 取决于 RPC 提供商的保留策略
  ❌ LLM Provider 日志: 取决于 Provider（优先选 zero-retention provider）
```

## 五、和之前设计的连接

| 模块 | 数据流关系 |
|------|-----------|
| Agent Profile | Profile 存 IPFS——公开可访问。不包括隐私数据 |
| 声誉系统 | 链上 Event——永久公开。task_hash 代替明文任务描述 |
| Escrow 流程 | 交付物存 IPFS——内容寻址，删除需 unpin |
| AI Oracle | Oracle 输入/output hash 上链——可验证但参数通过 hash 隐藏 |
| Prompt Injection 防护 | 第三方文档标记为 `<UNTRUSTED_CONTEXT>`——数据流向隔离 |

## 六、数据风险评估矩阵

| 数据 | 泄露影响 | 泄露概率 | 缓解 |
|------|----------|----------|------|
| 私钥 | 🔴 灾难 | 🟢 极低（不在 Agent 运行时内） | MPC / Smart Account |
| API Key | 🟠 高（可盗用额度） | 🟢 低（仅 HTTP header） | 加密 keystore + 限额 |
| 钱包地址 | 🟡 中（隐私泄露） | 🔴 高（链上公开） | 接受——链上地址天然公开 |
| 交易历史 | 🟡 中（行为画像） | 🔴 高（链上公开） | 接受——使用新地址管理隐私 |
| 用户输入 | 🟡 中（意图泄露） | 🟠 中（发 LLM Provider） | zero-retention provider |
| 记忆 | 🟠 高（偏好+行为） | 🟢 低（纯本地） | 本地加密存储 |
| 日志 | 🟡 中 | 🟢 低（纯本地） | 地址截断 |
