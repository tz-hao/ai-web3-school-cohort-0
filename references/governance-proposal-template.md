AI **没有**做以下工作：

**投票决定由你做出。本摘要只帮助你更快理解提案，不替代你的判断。**

## 模板设计原则

| 原则 | 在这个模板里怎么体现 |
|------|---------------------|
| AI 可以整理，不能建议 | 最后一段「AI 声明」明确列出 AI 没做什么 |
| 每条结论有来源 | 「来源标注」节 + 每条论据都附链接 |
| 两面都呈现 | 「支持理由」和「反对理由」分开且等权 |
| 暴露不确定性 | 「未回答问题」节——不知道就是不知道 |
| 可追溯可验证 | 链上交易、历史提案、reputation event 都有 explorer 链接 |
| 人做最终决定 | 模板全程不出现「建议投票」字样 |
