### Phase 4：Session Key ↔ AllowanceModule 映射

Safe 生态中 Session Key 的四个约束全部有链上对应：

- **限额** → `setAllowance(delegate, token, amount, resetTime)`
- **时效** → `resetTimeMin` + `removeDelegate()`
- **白名单** → 每 token 独立 allowance，没有的 token 不能花
- **可撤销** → `removeDelegate()` 一键 kill switch，不需要 delegate 同意

### 坑

1. Anvil fork 需要代理访问 Base RPC，纯本地链没有 Safe 合约部署地址
2. protocol-kit v5 ESM 导入 `import Safe from "..."` 拿不到类，需 `pkg.default`
3. `createSafeDeploymentTransaction` 依赖 `safe-deployments` 包有对应 chainId 的合约地址
