# Track Choice - Cobo

I choose **Cobo / Agentic Economy x Cobo Agentic Wallet** for SafePay Guard Wallet.

SafePay Guard Wallet fits this track because the project focuses on how an AI agent can operate a wallet only within a scoped policy boundary, instead of holding unrestricted private keys. The core workflow maps naturally to Cobo CAW / Pact: the agent proposes payments, trades, or resource purchases; Pact defines budget, allowed actions, time window, and human approval thresholds; the wallet infrastructure enforces those limits and records audit evidence. This project is especially aligned with Cobo because it treats agent autonomy as a controlled execution problem, not just an AI UX problem.
